Average Tweets

The graph on the left shows the average tweets I had received over the course of 5 minutes. Clearly, this shows that there is a strong-positive correlation between the two variables, and that they may have a directly proportional relationship.

The Twitter stream API is limited in that, depending on the level of the API you are using (higher levels can be acquired by paying tiwtter for the information). Since I am using the public level API, the web application can only receive around 1%-40% of the tweets that are sent out in one instance. Effectively, this means that I can receive about 0.48 tweets per second.
